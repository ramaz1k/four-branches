# 🏨 CozyStay — Hotel Booking Service

Современный и отзывчивый веб-сервис для поиска и бронирования отелей. Проект позволяет пользователям подбирать номера по различным критериям, просматривать подробную информацию об отелях и управлять своими бронированиями в реальном времени.

---

## ✨ Ключевой функционал

*   🔍 **Умный поиск:** Фильтрация отелей по городам, датам заселения, количеству гостей и ценовому диапазону.
*   🛏 **Детализированные карточки:** Просмотр галереи комнат, списка удобств (Wi-Fi, бассейн, завтрак) и отзывов.
*   📅 **Модуль бронирования:** Интерактивный календарь для выбора дат и мгновенный расчет итоговой стоимости.
*   📱 **Полная адаптивность:** Интерфейс идеально подстраивается под любые экраны — от мобильных телефонов до больших мониторов.

---

## 🛠 Стек технологий

Проект построен с использованием разделения логики на клиентскую и серверную части:

*   **Frontend:** HTML5, CSS3, Tailwind CSS (для быстрой и гибкой стилизации UI), JavaScript (ES6+).
*   **Backend (API):** Node.js / Express (обработка запросов, логика бронирования и валидация дат).
*   **База данных:** MongoDB / PostgreSQL (хранение данных об отелях, пользователях и заказах).
*   **Контроль версий:** Git & GitHub.

## 🚀 Быстрый запуск

Чтобы развернуть проект локально, выполните команду в терминале:

```bash
# 1. Клонируйте репозиторий
git clone [https://github.com/username/hotel-booking-system.git](https://github.com/username/hotel-booking-system.git)

# 2. Перейдите в директорию проекта
cd hotel-booking-system

# 3. Установите зависимости
npm install

# 4. Запустите проект в режиме разработки
npm run dev
