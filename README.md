## 🚀 Функционал

*   **Поиск и фильтрация:** Поиск отелей по городам, датам поездки, количеству гостей и ценовому диапазону.

*   **Личный кабинет:** Просмотр истории бронирований, управление активными заказами и профилем.
*   **Интерактивный интерфейс:** Современный, адаптивный дизайн (удобно пользоваться как с ПК, так и со смартфонов).
